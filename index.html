Customize Your Profile Card

How to Use:

  • Edit the fields above to customize the profile card in real-time
  • Click on a color to change the accent color
  • Choose a different style for the card
  • To use in your project, copy the HTML structure and link to style.css

For Developers:

You can also customize the card by editing the CSS variables in style.css:

:root {
    --primary-color: #4a6fa5;
    --card-bg: #ffffff;
    --text-color: #333333;
    --border-radius: 16px;
    --card-width: 340px;
}
